なお、「ちびAppearance Miku Ver1_5.pmx」は肩がsdefとなっており、変形には強い

いますが、上に強く曲げるとへこみが出来るようです。

「ファイル」→「インポート」　これで大丈夫のはず。

　ていますので、　ジョイントの設定を変更した上で、剛体の「頭a」は除去してしまって
　も良いかもしれません。さらに　モーフで格納するように出来れば良いと思いますが、
　この辺は各人の判断でどうぞ。
　おまけパーツ自体は他のモデルに流用していただいても問題ありませんが、そうなると
　テクスチャやスフィアを指定しなおすか、それらを削除して材質の色を髪などに合わせる
　必要があります。

■通常版（SDEF）とBDEF版の違い
ボーン変形のみ異なります。（髪やネクタイはどちらもSDEF変形です。）

通常…変形部がヒケにくいのですがｘ軸回転などでのエッジが目立ちやすいです。
　　　また、ボーン改造でデフォルトの親子関係が変わると、ウェイト破綻が起こる場合があります。

BDEF…比較的エッジは目立ちませんが、変形部分がヒケやすいです。ボーン改造する場合はこちらのデータがオススメ。
